- JWT

- MongoEngine

- MongoDB

  

## Installation

  

### Clone

```bash

$ git clone https://github.com/szatrio/tiny-wallet-api

$ cd tiny-wallet-api

```

---

  

### Create Environment Variable

```bash

$ mkvirtualenv tiny-wallet

$ pip install -r requirements.txt

```

---

### Start Development Server

```bash

$ flask run

```
